# 환전 요청 관리 프로젝트

## 프로젝트 소개

해당 프로젝트는 고객의 환전 요청을 관리하는 개인 학습 프로젝트입니다.

- 프로젝트 기간 : 11/26 ~ 11/29

## 프로젝트 목적

- N:M 연관 관계 CRUD 학습
- 영속성 전이 학습
- Validation, 전역 예외 처리 학습
- Post Construct 학습
- JPQL 학습
